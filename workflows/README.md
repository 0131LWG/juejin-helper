# Juejin Workflows 掘金工作流

## 开始使用

下载项目 或 [ clone the repo:](https://github.com/iDerekLi/juejin-helper)
```shell
curl https://codeload.github.com/iDerekLi/juejin-helper/tar.gz/main | tar -xz --strip=2 juejin-helper-main/workflows
cd workflows
```

安装依赖并运行脚本
```shell
yarn
yarn checkin
yarn seagold
```
